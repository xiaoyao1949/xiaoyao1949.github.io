组件 名称 用途

单选框 Radiobutton

文本框（多行） Text

菜单 Menu

滑块 Scale 允许通过滑块来设置一数字值
滚动条 Scrollbar 配合使用canvas,entry,listbox,text窗口部件的标准滚动条
框架 Frame 将几个组件组成一组
