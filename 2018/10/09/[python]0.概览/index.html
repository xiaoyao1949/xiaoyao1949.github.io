python概览

发表于 2018-10-09 | 更新于 2019-03-11 | 分类于 python基础

python语言的特点，计算生态，执行方式以及为什么需要python。

python是什么?

  1. 通用语言（全领域）
  2. 脚本语言(解释型语言)
  3. 动态语言（不适用显式数据类型声明）
  4. 强类型语言（不允许不同类型相加）
  5. 胶水语言
  6. 跨平台语言
  7. 多模型语言

语法简洁:简单易读,接近伪代码,代码量少,容易调试,快速开发

开源生态:可扩展功能强,避免重复造轮子

语法简介+开源生态=超级高产

python计算生态

  1. 没有顶层设计,开源项目野蛮生长,自然选择,高质量库众多
  2. 以功能为单位,以开源项目为形式
  3. 竞争发展
  4. 相互依存,依存发展或逐级封装
  5. 社区庞大,迅速更迭

编程语言执行方式:

  1. 编译执行

    静态语言如c/c++/Java语言,源代码通过编译器生成目标代码,目标代码通过链接生成可执行程序,引导可执行程序创建进程,输入数据产生结果.

    一次性翻译,之后不再需要源代码

    编译阶段可以确定内存空间，基本程序逻辑等

    编译过程没有时间限制,优化更充分,执行速度更快

    编译后的程序可以在同系列系统中直接执行,不需要执行环境

  2. 解释执行

    如Python/PHP/JavaScript语言,源代码与数据输入解释器,解释器将源代码逐条转换成目标代码同时逐条运行.

    每次程序运行时都逐条解析源代码并执行

    执行过程需要源代码,程序维护更灵活

    不需要在编程阶段确定更多信息，根据代码执行的实际情况决定一切。

    过在不同系统中配置执行环境,可以实现源代码的跨平台执行
